(static) extend(pElement, pName, pFunction, pType)

Allows additional function to be binded to the element type.
Parameters:
Name Type Description
pElement string | Array.<string> the HTML element to bind the function
pName string the name of the additional function
pFunction function the function body
pType integer (Optional)specific the type to be binded with the function.
If it is "element" or "collection", the function will be binded directly to the object.
If it is "both", it behaves the same for single element object but will wrap the function with a forEach loop to apply the function to all child of a collection object when called`
